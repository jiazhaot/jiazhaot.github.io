# INFO20005_2022_SM1: Advent Games UI Redesign Project

## Project Overview

This project focuses on the front-end UI redesign of **Advent Games** (https://www.adventgames.com.au/), a premier destination for board game enthusiasts. Notably, this project prioritizes front-end aesthetics and usability without implementing back-end database integration or payment processing functionality.

## Key Objectives

1. **Aesthetic Enhancement**: Overhaul the visual design with a cohesive color palette, typography system, and layout that reflects the fun, immersive nature of board gaming while maintaining professional clarity.
2. **Responsive Design**: Ensure seamless interaction across devices (desktop, tablet, mobile) with adaptive layouts that preserve functionality and visual appeal.

## Technical Stack

- HTML: Semantic markup for accessibility and structure.
- CSS (with Flexbox/Grid): Responsive layout and styling.
- JavaScript: Interactive elements (carousels, filters, modals) and dynamic content updates.

## Why This Matters

Board games foster connection, creativity, and critical thinking—and the digital experience of browsing and discovering them should be just as engaging. By reimagining Advent Games’ UI, this project bridges the gap between the physical joy of board gaming and the convenience of online shopping. A well-designed front-end not only attracts users but also encourages them to explore, engage, and return—ultimately supporting Advent Games’ mission to bring people together through tabletop experiences.

## Future Enhancements (Post-Project)

- Integrate back-end database (e.g., MySQL, Firebase) to dynamically load product data.
- Implement user authentication and personalized recommendations.
- Add functional "Add to Cart" and checkout flows with payment gateway integration (e.g., Stripe, PayPal).
- Incorporate user reviews, ratings, and wishlist features.
- Integrate a blog or community section for game tips, event announcements, and user stories.
